# angularjs-feed
### Introduction
This feed application renders RSS feeds using [AngularJS](https://angularjs.org/).

### Dependencies
* [SKY UX](https://skyux.developer.blackbaud.com/)
* [rss2json](https://rss2json.com/)
* [Bootstrap](https://getbootstrap.com/)
